Stroke Analyzer¶

stopeight-clibs-legacy¶

As of 09/2015 only python version 2.7 has been tested.

Note: Needs qt4-core and qt4-gui:

$ sudo apt-get install libqt4-core
$ sudo apt-get install libqt4-gui

stopeight-clibs-legacy. Get:

$ git clone https://github.com/specpose/stopeight-clibs.git

Run qmake to generate the Makefile, then compile, install and add to library path:

$ cd /path/to/where/stopeight-clibs/legacy
$ qmake-qt4 -o ./Makefile legacy.pro
$ make
$ sudo make install
$ sudo ldconfig

Build and install the python API for pen-strokes:

$ cd /path/to/where/stopeight-clibs/legacy-wrappers
$ python setup.py build
$ sudo python setup.py install

Analyzing pen-strokes from a 2D input device.¶

Pen stroke test data can be found in the repository stopeight-clibs. As you can see you have now loaded the raw data of a pen-stroke. Any arc-wise connected series of 2D points can be loaded but you have to provide your own GUI at the time being. The sample data has been hand-drawn on a Wacom Cintiq 21UX and plain mouse using the editor binary.

It has to be extracted from here:

$ cd /path/to/where/stopeight-clibs/tests
$ tar xvfz v0-1-0.tar.gz

Load a file with pen-stroke test data:

>>> import stopeight_clibs_legacy
>>> line = stopeight_clibs_legacy.parse_file('/path/to/where/FILENAME.sp')
>>> print line

To extract the feature points from the pen-stroke:

>>> points = stopeight_clibs_legacy.stroke_parallel(line)
>>> print points
