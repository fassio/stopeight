Stopeight¶

Instructions have been tested on Ubuntu 18.04/bionic.

Installation¶

Minimal Requirements (main repository):

$ sudo apt-get install python cmake build-essential

If you want to run the tests for the legacy algorithm, install Qt5 (main repository).

$ sudo apt-get install qtbase5-dev

The stopeight repository contains stopeight-clibs (Github):

$ git clone --recurse-submodules https://github.com/specpose/stopeight.git
$ cd /path/to/where/stopeight

Build and install stopeight (easy_install):

$ easy_install-3.6 --user virtualenv
$ virtualenv ~/stopeight-test
$ ~/stopeight-test/bin/python setup-easydev.py develop

Try the editor for a start:

$ ~/stopeight-test/bin/python -m stopeight.util.editor.dispatch

Uninstall:

$ ~/stopeight-test/bin/python setup-easydev.py develop --uninstall
$ rm -R ~/stopeight-test
$ rm -Rf /path/to/where/stopeight
$ rm -R ~/.stopeight
