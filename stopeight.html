Installation¶

stopeight-clibs¶

Download and install anaconda

You have to add cmake and qt dependencies manually and install stopeight-clibs:

$ conda install -c specpose cmake qt stopeight-clibs

For installation from source refer to Build.

stopeight¶

Note: You need to install stopeight-clibs before installing stopeight or the python extensions are not compiled.

Python 2: PyQt5 has been removed for Python 2. You have to install it manually from Riverbank or your distro. Also you need a recent gcc:

$ sudo apt-get install python-pyqt5 gcc-6
$ pip install numpy future funcsigs matplotlib pybind11>=2.2
$ pip install --user --no-deps stopeight

Python 3:

$ pip install --user stopeight

Eventually you will need the server module, test scripts and test data:

$ git clone https://github.com/specpose/stopeight.git
