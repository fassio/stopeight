Comparator
==========

Dependencies
------------

As of 09/2015 only python version 2.7 has been tested.

Get numpy on ubuntu::

    $ sudo apt-get install python-numpy

.. automodule:: stopeight.multiprocessing.pooling
   :members:

Is using the Python multiprocessing module. You can test if your Python installation and the underlying OS support this by running from within a Python shell::

    >>> from multiprocessing import synchronize
