Applications
============

It can be applied to a lot of different applications:

* :doc:`stroke-analyzer`
   Generic frontend for pen-strokes.
   stopeight-clibs-legacy: Partially working.

   Example: Recognising Asian characters. (See http://www.symboloh.com)
* Analyzing a time-based series of Function Graph data.
   Turns a Function Graph into a Vector Graph.
   stopeight-clibs-grapher: Working.

   Extracts feature points. Resolution independent version of legacy.
   stopeight-clibs-analyzer: Not yet implemented.

   Example: Frontend for non-realtime signal analysis.

In different fields:

* Pen-Stroke UI
* Signal Analysis
* Data Visualisation
