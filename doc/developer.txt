GitHub & Sphinx
===============

In order to rebuild the docs from the project directory, do::

    $ sudo apt-get install python-sphinx
    $ cd doc/
    $ make html
    $ cd ../
    $ git checkout gh-pages
    $ git add doc/_build/html/*
    $ git commit -m "Documentation vX.Y.Z"
