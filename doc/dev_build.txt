Build
-----

Requirements (can also be satisfied via conda :doc:`stopeight`)::

    $ sudo apt-get install cmake build-essential qtbase5-dev

The stopeight repository contains stopeight-clibs::

    $ git clone --recurse-submodules https://github.com/specpose/stopeight.git
    $ cd /path/to/where/stopeight

If you prefer the latest development version of stopeight-clibs::

    $ rm -R stopeight-clibs
    $ git clone --recurse-submodules https://github.com/specpose/stopeight-clibs.git

Build and install stopeight-clibs::

    $ cd stopeight-clibs
    $ mkdir build && cd build
    $ cmake ../
    $ make && sudo make install

Note: You may need to manually reinstall after a rebuild::

    $ sudo cp libstopeight-clibs-* /path/to/where/your/syslibs
    $ sudo ldconfig

Build and install stopeight::

    $ cd /path/to/where/stopeight

Python 2: Optional BuildExt modules are not supported, Qt5 is mandatory. PyQt5 has been removed for Python 2. You have to install it manually from `Riverbank <https://www.riverbankcomputing.com/software/pyqt/download5>`_ or your distro. Also you may need a recent gcc::

    $ git checkout Python2
    $ git pull
    $ sudo apt-get install qtbase5-dev python-pyqt5 gcc-6
    $ pip install -r requirements-python2.txt
    $ cd /path/to/where/stopeight/..
    $ pip install --user --no-deps stopeight

Python 3::

    $ python3 setup.py install --user
